## 📂 Projects

### 🔌 [ES_DGS - D-Link モジュール棚卸ツール](https://github.com/Konno8329/ES_DGS)
複数の D-Link スイッチに SSH 接続し、全ポートの SFP モジュール情報を一括収集して Excel に出力するツール。

**背景**: ネットワーク更改案件で、数十台のスイッチのモジュール使用状況を手作業で調査する必要があった。台数が増えると時間がかかりミスも発生しやすいため、Python × Netmiko で自動化。在庫把握・発注数量の確認・設計資料の作成に活用した。

`Python` `Netmiko` `openpyxl` `D-Link DGS`
